# Jeff Chen - Portfolio

此為個人作品集範例，示範專案導向的前端靜態網站。內容包含作品卡片、彈窗說明與響應式樣式。

本地預覽

1. 在本機打開專案資料夾
2. 直接以瀏覽器開啟 `index.html` 或使用簡易 HTTP server：

```powershell
# Windows: 使用內建 PowerShell 伺服器
cd c:\Users\owner\Desktop\my-profile-website
python -m http.server 8000
# 然後瀏覽 http://localhost:8000
```

自訂內容

- 將個人照片放在 `image/profile.jpg`（若無則會自動隱藏圖片區塊）。
- 更新 `index.html` 中的專案卡片（`data-*` 屬性）連結到你的 GitHub repository。
- 可在 `css/style.css` 調整主題色與字型。
